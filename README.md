# Promise of the Crown

## Game Description
**Promise of the Crown** is a 2D fantasy platformer adventure game where the player embarks on a journey through a mystical forest to fulfill a royal promise.  
Players explore hand-crafted levels, jump across platforms, and progress through the environment to achieve the main objective of the game.

The game emphasizes atmosphere, simple controls, and exploration-based gameplay.
